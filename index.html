API Testing Report

This project provides an API testing report for the Pet Store. The report details test cases for various API endpoints, including functional, negative, and boundary value tests. Testing covers operations such as getting pet information, updating pet information, searching pets by status, deleting a pet, and creating orders. The goal of the project is to ensure the reliability and correct operation of the API in various scenarios.

Selenium & Jest test automation - Webshop

The project aims to automate testing of the Waterstones online store using Selenium and Jest. It includes tests that test searching for products using keywords, sorting and filtering products, adding products to cart, and checking the cart. The tests are written according to the described test scenarios and follow the principle of the Page Object pattern. The main goal is to ensure the functionality and reliability of the online store.

Refactoring - Car Rental Company Price Calculator

The project focuses on refactoring the code for the car rental price calculator to bring it up to clean code standards. Includes fixing bugs and adding missing code to meet new business requirements. The program takes into account various factors such as the driver's age, vehicle type, season and rental duration to calculate the final rental cost. The project also includes TDD to add new functionality to differentiate prices on weekdays and weekends.
